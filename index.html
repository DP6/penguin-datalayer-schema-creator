Como utilizar

Preencha os campos com as informações que serão validadas no datalayer, caso tenha preechido errado poderá fazer a correção, e depois de tudo preechido faça o download do Json Schema para poder compará-lo com o datalayer, você também terá a opção de copiar as informações inseridas como texto.

Json Schema

O JSON Schema é uma estrutura que permite a validação de documentos JSON. Esta estrutura é utilizada no projeto pois permite a declaração dos formatos de dados esperados dentro da camada de dados.

Tipos Suportados

Os seguintes tipos de dados são suportados:

  • String
  • Number
  • Boolean
  • Object
  • Array

Estrutura do JSON Schema

A estrutura a seguir é um exemplo de um JSON Schema:

          
            
            
            {
              "$schema": "",
              "title": "Schema example",
              "array": {
                "$id": "#/properties/schema",
                "type": "array",
                "items": [
                  {
                    "type": "object",
                    "properties": {
                      "event": {
                        "type": "string",
                        "enum": ["teste"]
                      },
                      "key1": {
                        "type": "object",
                        "properties": {
                          "key1_sub1": {
                            "type": "number",
                            "enum": [10]
                          },
                          "key1_sub2": {
                            "type": "string",
                            "pattern": "teste|test|.*"
                          },
                          "key1_sub3": {
                            "type": "string",
                            "enum": ["producao"]
                          },
                          "key1_sub4": {
                            "type": "boolean",
                            "enum": "desktop|mobile|msite"
                          }
                        },
                        "required": ["key1_sub1", "key1_sub2", "key1_sub3", "key1_sub4"]
                      }
                    },
                    "required": ["event"]
                  }
                ]
              }
            }
